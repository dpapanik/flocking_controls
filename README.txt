How to run in Mac

1. Open terminal window
2. Change Directory to the location in your computer where you have this folder
	Example: cd /Users/dpapanik/Documents/.../flocking_basic
3. Type python -m http.server
	See more instructions here: https://github.com/processing/p5.js/wiki/Local-server
4. Open your web browser
5. Type http://localhost:8000/
6. To kill the python server type ctrl+C
